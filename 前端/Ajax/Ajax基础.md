# Ajax基础

## 简介

AJAX 全称为 Asynchronous JavaScript And XML，就是异步的 JS 和XML。通过 AJAX 可以在浏览器中向服务器发送异步请求，最大的优势：<font color='red'>无刷新获取数据</font>。AJAX 不是新的编程语言，而是一种将现有的标准组合在一起使用的新方式。



## 特点

### 优点

- 可以无需刷新页面而与服务器端进行通信。
- 允许你根据用户事件来更新部分页面内容。

### 缺点

- 没有浏览历史，不能回退。
- 存在跨域问题（同源）。
- SEO 不友好（数据不是直接在页面上，爬虫工具无法爬取到数据）。
